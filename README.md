# 📦 Interface de Produtos - Desenvolvimento Web (2024)

Este projeto foi desenvolvido em **2024** como parte da disciplina de **Desenvolvimento Web**. O objetivo foi criar uma interface moderna e funcional para a exibição de produtos, aplicando conceitos avançados de estilização e organização visual.

---

### 🚀 Sobre o Projeto

A **Interface** foca em apresentar um catálogo de produtos de forma limpa e atrativa, utilizando técnicas de design responsivo para garantir uma boa experiência de navegação.

**Principais Funcionalidades:**
* **Layout Adaptável**: Estrutura organizada para exibição de múltiplos produtos em grade.
* **Cards de Produto**: Estilização detalhada para imagens, nomes, descrições e preços.
* **Interatividade**: Feedback visual ao passar o mouse (hover) sobre os elementos de ação.
* **Design Moderno**: Uso de sombras, bordas arredondadas e tipografia selecionada para um aspecto profissional.

---

### 🛠️ Tecnologias Utilizadas

O projeto foi construído com foco em performance e simplicidade, utilizando a "Vanilla Stack":

* **HTML5**: Estruturação semântica de toda a página e dos componentes de produto.
* **CSS3**: 
    * Flexbox e Grid para o posicionamento dos elementos. 
    * Customização de botões e formulários.
    * Manipulação de cores e efeitos de profundidade.

---

### 🎨 Como Visualizar

1. Faça o download ou clone este repositório.
2. Acesse a pasta do projeto.
3. Abra o arquivo `interface2.html` em seu navegador de preferência.
